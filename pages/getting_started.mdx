# Primeros Pasos

### ¿Cómo empiezo a cobrar en criptomonedas?

Estos pasos son los que hay que seguir para comenzar:

1. Creá tu cuenta de **talo**.
2. Realizá la integración para recibir pagos.
3. Probá la integración en modo sandbox.
4. Deployá la integración a producción.
5. Monitoreá los pagos desde el tablero de **talo**.

## 1. Creá tu cuenta de **talo**

Para poder crear una cuenta de **talo** solo necesitás tener una cuenta de email. <Link href="https://talo.com.ar/signup"> Registrate </Link> ingresando tu email y una contraseña y luego verificá tu e-mail para poder comenzar a recibir pagos.

## 2. Realizá la integración para recibir pagos

El siguiente paso es realizar la integración con la API de **talo** para comenzar a recibir los pagos. Pasá por nuestra <Link href="/guides/payments">guía de pagos</Link> para saber el paso a paso de la integración.

## 3. Probá la integración en modo sandbox

Una vez que lograste completar el flujo de pago como se indica en la guía de la integración de pagos ya es momento de comenzar con las pruebas. **Talo** cuenta con un modo sandbox para poder realizar las pruebas de forma segura y sin riesgo.
La <Link href="/sandbox">sección del sandbox</Link> tiene más información sobre este modo. Para realizar las pruebas podés seguir el paso a paso de la <Link href="/guides/testing">guía para hacer pruebas</Link>.

## 4. Deployá la integración a producción

¡Ya casi terminás! Lo único que falta es que una vez conforme con las pruebas realizadas deployes la integración a tu sistema en producción para que lo puedan comenzar a utilizar tus clientes cuánto antes.

## 5. Monitoreá los pagos desde el tablero de **talo**

¡Listo! Ya sos parte de la transición al mundo de las criptomonedas. Para poder monitorear y llevar registro de los pagos que se realizan en criptomonedas podés ingresar en tu tablero de **talo** y extraer de ahí los datos que necesites.
