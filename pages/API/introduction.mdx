# Introducción

En talo exponemos los siguientes métodos de nuestra API para que los comercios puedan integrar en sus sistemas internos el cobro en criptomonedas.

## API de Pagos

La API de Pagos expone la funcionalidad principal del flujo de cobro. Consta de 2 funcionalidades principales:

- <Link href="/API/payment_api">Crear un pago</Link>
- <Link href="/API/payment_api">Obtener información sobre un pago</Link>

## API de Usuarios

La API de Usuarios permite obtener tokens para poder autenticarse con el sistema de talo. Para obtener información sobre un pago por ejemplo se requiere utilizar el método de la API con un token de autenticación. La funcionalidad provista por la API de usuarios es:

- <Link href="/API/users_api">Obtener tokens de autenticación</Link>
