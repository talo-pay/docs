# Sandbox

Para poder hacer pruebas con la API sin tener que estar gastando plata real, en _talo_ creamos el modo sandbox. Lo podés acceder creándote una cuenta en [`Talo Sandbox`](https://sandbox.talo.com.ar), con tus mismas credenciales que en tu cuenta real, u otras completamente nuevas.

## Enviar dinero a los pagos creados

Ahora que ya estamos en el modo Sandbox, lo único que falta es arrancar a crear órdenes de pago y transferirles dinero para probar todo el ciclo del proceso de pago.

### 1.Billetera para Sandbox

La forma más facil de enviar y recibir cripto es desde una billetera descentralizada como [Metamask](https://metamask.io/). Si ya tenés descargada la billetera y una cuenta creada, solo te falta agregar la red de prueba e ir a la "canilla", que te proveerá fondos de prueba.

### 2.Agregar Redes de prueba

Por default, en billeteras como Metamask al arrancar ya está configurada la red de Ethereum principal, también llamada _Ethereum Mainnet_. Para poder realizar transacciones en la red de prueba hay que agregarla manualmente a la billetera. Clickeá en el siguiente boton para que se cargue la información predeterminada de la red de prueba de BSC.

<AddNetworkButton />

Alternativamente, si tuviste problemas agregandolo de esta forma, podes agregarla manualmente vos siguiendo el tutorial en la pagina de [Binance Smart Chain](https://academy.binance.com/es/articles/connecting-metamask-to-binance-smart-chain)

### 3.Conseguir fondos de prueba

Para poder probar protocolos y realizar transacciones en un ambiente de prueba existen sitios llamados faucets que disponibilizan tokens y monedas para utilizar en las testnets. Como pertenecen a las redes de pruebas estos tokens no tienen ningún valor real, simplemente nos permiten probar interacciones de forma segura.
