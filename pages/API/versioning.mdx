# Versionado

La API de talo está en constante evolución para satisfacer lo mejor posible los requisitos de los usuarios.
Es importante tener en cuenta que una actualización podría modificar los parámetros esperados en las solicitudes así como el formato de las respuestas.

Para asegurar la retrocompatiblidad el versionado de la API de **talo** no se verá en los endpoints.
El versionado se realizará sobre los objetos que se reciben y devuelven en las solicitudes.
